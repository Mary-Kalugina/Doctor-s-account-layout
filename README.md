<h3>To start the project:</h3><br>
1. Clone the project<br>
2. Run `npm install`<br>
3. Run `npm run dev`<br>

<h3>#Build and infrastructure:</h3><br>
- Webpack 5 + HMR<br>
- TypeScript + React 18<br>
- SCSS + Autoprefixing <br>
